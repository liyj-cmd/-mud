# WRITING STYLE（文案与叙事风格）

## 目标风格
- 偏传统武侠语感
- 句子简洁，不堆辞藻
- 以动作与选择推动剧情，不靠大段旁白

## 文案层级
- 地点描述：1-2 句，强调气氛与功能
- NPC 台词：短句，体现立场与性格
- 事件描述：给冲突，不提前给结论
- 结果文本：明确代价或收益

## 角色语气建议
- 掌门/长老：稳重、克制
- 游侠/江湖客：直接、机敏
- 官府角色：规矩、务实
- 商人角色：利益导向但不脸谱化

## 禁止写法
- 现代网络口语穿帮
- 纯信息堆砌、没有动作
- 结果文本与实际效果不一致

## 编写流程建议
1. 先定义选择与效果
2. 再写描述与台词
3. 最后核对世界观一致性
