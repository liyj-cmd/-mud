# 武侠单机 MUD（极简原型）

这个项目是一个极简的武侠单机 MUD 原型，用于演示“时间轴”机制：
玩家每次行动会推进时间，其他人物也会在同一时间刻内行动。

## 运行方式

```bash
python mud.py
```

## 当前可用指令

- `look`：查看当前场景
- `go <方向>`：移动（north/south/east/west）
- `wait`：原地等待
- `status`：查看角色状态
- `help`：显示帮助
- `quit/exit`：退出游戏
